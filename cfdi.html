Visor CFDI

Visualiza archivos de formato XML en una vista mas amigable. Cuando el archivo es un CFDI puedes validar el estatus ante el SAT con un solo clic.

Características

  • Consulta y revisión de CFDI.
  • Acceso sencillo para usuarios registrados.
  • Versión gratuita para comenzar a evaluar el sistema.

Condiciones de uso

  • El usuario debe registrarse antes de usar la aplicación.
  • Puedes mantener gratis 30 archivos XML.
  • Precio mensual de 232 pesos, pagaderos por transferencia o tarjeta bancaria.
  • La versión por suscripción permite mantener hasta 10,000 archivos XML en la cuenta.
