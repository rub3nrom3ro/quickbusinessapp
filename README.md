# Quick Business App site starter

Este directorio contiene un sitio estático inicial para promover tus sistemas mensuales.

## Qué incluye
- Landing de una página
- Diseño responsive básico
- Secciones para sistemas, beneficios y contacto

## Cómo usarlo
1. Sube `index.html` y `styles.css` a tu hosting.
2. Conecta el dominio `quickbusinessapp.com`.
3. Reemplaza el correo de contacto por el tuyo real.
